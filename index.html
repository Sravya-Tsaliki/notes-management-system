📝 Welcome to Notes Management System

Organize your study notes, personal reminders, and work tasks efficiently using this Notes Management System.

✨ Features

  • ✅ Add Notes
  • ✅ Edit Notes
  • ✅ Delete Notes
  • ✅ Search Notes
  • ✅ Categories (Study, Personal, Work)
  • ✅ Description Text Colors
  • ✅ Background Colors
  • ✅ Pin Important Notes
  • ✅ Dark Mode
  • ✅ Date & Time Tracking
  • ✅ Local Storage Support
  • ✅ Mobile Responsive Design

📖 How To Use

  1. Enter a Note Title.
  2. Enter the Note Description.
  3. Select a Category.
  4. Select Description Text Color.
  5. Select Note Background Color.
  6. Click Add Note.
  7. Use Search to find notes quickly.
  8. Pin important notes to keep them at the top.
  9. Use Dark Mode for comfortable viewing.
  10. Your notes are automatically saved.
  11. Refresh the page without losing your notes.

💾 Data Storage

This application uses browser Local Storage. Notes are saved automatically and remain available even after refreshing or reopening the browser.
