# Notes Management System

A Notes Management System built using HTML, CSS, and JavaScript.

## Features
- Add Notes
- Edit Notes
- Delete Notes
- Search Notes
- Categories
- Pin Notes
- Dark Mode
- Local Storage
- Responsive Design

## Technologies Used
- HTML
- CSS
- JavaScript
